# Payroll Management System (JavaFX)

## Description
This is a JavaFX-based payroll management system that allows managing employee data and salary processing.

## Features
- Employee management
- Payroll calculation
- GUI interface using JavaFX

## Technologies Used
- Java
- JavaFX
- Maven

## How to Run
1. Clone the repo
2. Open in IntelliJ or NetBeans
3. Run HelloApplication.java

## Note
Make sure JavaFX is properly configured.
